Mandelbrot to Julia

Mandelbrot set and Julia set are in beautiful relationship. Each point of the Mandelbrot set can be used as parameter for the Julia set. Therefore every Julia set is unique and is related exactly to one point! Click to Mandelbrot set and try it! And don't forget to change filters. :) This topic is well explained this youtube video.
